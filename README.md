# 本地化思维导图工具

这是一个使用 React + TypeScript + Vite 搭建的本地化思维导图工具基础工程。当前阶段只包含最小可运行页面，后续会逐步实现思维导图编辑、文件存储、导入导出等能力。

## 环境要求

- Node.js 18 或更高版本
- npm 9 或更高版本

## 安装依赖

```bash
npm install
```

## 启动开发服务器

```bash
npm run dev
```

启动成功后，在浏览器打开终端中显示的本地地址，通常是：

```text
http://localhost:5173/
```

## 构建验证

```bash
npm run build
```

## 当前已包含内容

- Vite + React + TypeScript 基础工程
- 基础页面标题：本地化思维导图工具
- 基础按钮：新建思维导图

- Markdown 备注
