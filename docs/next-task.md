# 下一步任务建议

## 优先建议

1. Tauri 环境补齐与桌面端验证：
   - 安装 Rust / Cargo。
   - 在 Windows 上运行 `npm run tauri:dev` 和 `npm run tauri:build`。
   - 验证桌面窗口启动、加载当前 Vite 应用、保存/打开 `.lmind`、导入导出和图片导出。
   - 继续验证 macOS、Linux、统信 UOS、银河麒麟和国产 CPU 架构。
2. 节点拖拽体验补强：
   - 验证 1000 节点导图拖拽响应。
   - 优化拖拽过程中的性能和选中态。
   - 评估是否需要吸附、辅助线或简单避让。
3. 多选能力继续增强：
   - 增加框选多个节点。
   - 增加批量移动、批量样式编辑。
   - 评估多选后的复制 / 剪切 / 粘贴。
4. 自动化测试继续补充：
   - 增加 UI 组件测试或端到端测试。
   - 覆盖保存、打开、导入导出、插件启停、模板、主题、拖拽、多选、右键菜单等主流程。
5. UI 细节打磨：
   - 检查小屏和高 DPI 表现。
   - 优化顶部工具区密度。
   - 优化弹窗、右键菜单、备注面板和长文本溢出。

## 后续功能

- 节点复制 / 剪切 / 粘贴。
- 拖拽节点调整层级与顺序。
- 框选多个节点和批量操作。
- 更完整的主题和线条样式配置。
- 插件签名、插件目录和沙箱执行能力。
- 桌面端真实安装包 Release。
