# 下一步任务建议

## 优先建议

1. 桌面端真实打包：
   - 选择 Tauri 或 Electron 最小方案。
   - 保持 Web 版和 GitHub Pages 不受影响。
   - 验证 Windows、macOS、Linux / 信创系统。
2. 官方模板库：
   - 设计官方模板数据结构。
   - 增加不可删除的内置模板。
   - 与现有自定义模板列表共存。
3. 更完整自动化测试：
   - 增加 UI 组件测试或端到端测试。
   - 覆盖保存、打开、导入导出、插件启停、模板、主题等主流程。
4. 性能优化：
   - 准备 1000 节点测试文件。
   - 测量打开、编辑、导出耗时。
   - 评估大导图渲染和 Markdown 预览性能。
5. UI 细节打磨：
   - 检查小屏和高 DPI 表现。
   - 优化工具区拥挤问题。
   - 优化弹窗、面板和长文本溢出。

## 后续功能

- 节点复制 / 剪切 / 粘贴。
- 拖拽节点调整层级与顺序。
- 框选多个节点和批量操作。
- 更完整的主题和线条样式配置。
- 插件签名、插件目录和沙箱执行能力。
