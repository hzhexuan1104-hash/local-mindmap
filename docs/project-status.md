# 项目状态

- React + TypeScript + Vite 基础工程。

- 节点折叠 / 展开。
- 主题系统，内置 5 套主题。
- 插件管理系统基础版：
  - 本地 JSON 插件安装。
  - 插件启用、禁用、卸载。
  - 3 个内置示例插件。

## 本次构建结果

- `npm run build` 通过。
- Vite 提示打包后主 chunk 超过 500 kB，这是体积提醒，不是构建错误。

## 安全状态

- 当前插件系统不执行第三方 JS。
- 不使用 `eval` 或 `new Function`。
- 不加载远程脚本。
- 插件数据保存在 localStorage。
- 用户文件仍通过本地浏览器文件能力导入导出。
