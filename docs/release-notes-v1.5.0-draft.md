# v1.5.0 发布说明（草案）

## 版本主题

v1.5 聚焦“画布优先的高质感极简 UI”。第一批完成界面信息架构和布局重构，第二批以用户提供的参考 UI 图为方向，完成浅色视觉系统与核心界面质感精修。

## 第一批：基础结构

- 顶部收纳为文件、编辑、插入、视图、导入导出、更多六组菜单。
- 左侧调整为窄栏与可收起资源面板。
- 右侧调整为样式、备注、信息 Tabs 检查器。
- 画布扩展为主视觉区域，控制条移动到右下角。

## 第二批：视觉精修

- 建立应用背景、画布、表面、边框、文本、强调色、危险色、阴影、圆角、控件高度和面板宽度视觉变量。
- 顶部精修为轻量桌面标题栏，清晰区分产品信息、当前导图、菜单和高频操作。
- 左侧精修纯图标窄栏、搜索框、当前文件卡片、筛选与模板轻卡片。
- 右侧精修分段 Tabs、主题 / 节点 / 类型分组、备注编辑和轻量信息行。
- 画布精修暖白纸感、极浅点阵、胶囊控制条和低干扰留白。
- 节点精修中心主题层级、普通节点弱边框与微阴影、柔和连线及统一选择 / 拖拽反馈。
- 统一顶部菜单、右键菜单、按钮、输入框、下拉框、Tabs、弹窗和危险操作样式。

## 兼容性与边界

- 本次不新增业务功能。
- 不修改 `.lmind` 基础结构。
- 不修改 Markdown、Excel、JSON、图片、TXT 导入导出格式。
- 不修改节点类型包和模板包格式。
- 不新增 npm 依赖，不修改 `package.json` 或 `package-lock.json`。
- 不修改 Tauri / Windows 桌面端代码，不影响 v1.4.1 Windows 修复。
- 不加入云同步、登录、协同、插件市场或完整深色主题重构。

## 验证

- `npm run build`：通过。
- `npm run test`：通过，17 个测试文件、142 个测试。
- `npm run dev`：本地页面正常打开，无运行时 error。
- 已手动检查菜单开关、左右面板、Tabs、右键菜单、画布平移、Shift 框选、root position 拖拽和工具条文字选择保护。

## 发布前仍需

- Windows Tauri 窗口与安装包视觉回归。
- 2K / 4K、高 DPI 和不同系统字体环境检查。
- `.lmind`、节点类型包、模板包与全部导入导出流程的桌面端手动回归。
