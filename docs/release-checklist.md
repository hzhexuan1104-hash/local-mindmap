| 风险 | 影响 | 处理建议 |

| 自动化测试以纯函数为主 | UI 回归仍依赖手动测试 | 后续补端到端测试 |
| 节点拖拽为基础版 | 不包含复杂避让或层级调整 | 后续评估拖拽调整层级、框选和智能排版 |
