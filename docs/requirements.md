一、文档概述
1.1 文档目的
本文档明确本地化思维导图工具的功能范围、业务规则、技术约束与验收标准，作为产品设计、开发、测试及验收的核心依据。
1.2 项目范围
本产品为纯本地化离线运行的思维导图工具，无云端依赖，覆盖市面主流思维导图基础能力，并新增模板管理、自定义节点类型、Markdown 备注、插件化导入导出等特色功能，适配 Windows、信创操作系统、macOS 三大平台。
1.3 目标读者
产品经理、UI/UX 设计师、开发工程师、测试工程师、项目管理人员。
1.4 术语定义
•节点类型：用户自定义的节点样式与内容模板组合，通过图标、颜色等视觉特征区分节点属性。
•信创版本：适配国产操作系统（统信 UOS、银河麒麟）及国产 CPU 架构的软件版本。
•所见即所得：Markdown 编辑过程中实时渲染最终格式效果，无需切换预览模式。

二、项目整体说明
2.1 产品定位
面向个人与企业用户的轻量化、可定制、全本地化思维导图工具，兼顾通用思维导图效率与个性化扩展能力，满足信创环境下的离线使用需求。
2.2 核心目标
1.覆盖主流思维导图工具的基础编辑能力，降低用户学习成本
2.通过自定义节点、模板体系实现差异化创作效率提升
3.实现 Markdown 富文本备注的实时渲染，强化内容承载能力
4.插件化架构支持导入导出格式扩展，适配多元数据场景
5.全平台原生适配，Windows / 信创 /macOS 体验一致、文件兼容
2.3 运行环境
平台类型	适配要求
Windows	Windows 10 及以上版本，x86/x64 架构
信创系统	统信 UOS 桌面版、银河麒麟桌面版；适配飞腾、鲲鹏、龙芯架构
macOS	macOS 11.0 及以上版本，Intel/Apple Silicon 架构
2.4 存储模式
•纯本地文件存储，无任何云端上传、同步逻辑
•原生存储格式为标准 JSON，文件后缀可定义为 .lmind
•文件可自由拷贝、分享，打开即可完整还原所有样式与配置

三、功能需求规格
3.1 基础思维导图核心功能（通用标配能力）
本模块为市面思维导图工具通用基础能力，作为产品底座实现：
1.节点操作：新增中心主题、子节点、同级节点；删除节点；双击编辑节点文本；节点复制 / 剪切 / 粘贴
2.结构调整：拖拽节点调整层级与顺序；支持自动布局与自由布局切换；连接线自动跟随节点位置更新
3.画布操作：画布缩放、平移；一键居中；全屏展示；框选多个节点批量操作
4.样式编辑：节点字体、字号、文字颜色、背景色、边框样式配置；连接线线型、颜色、粗细配置
5.视图控制：单个节点折叠 / 展开；一键展开全部 / 折叠全部分支
6.操作历史：多级撤销、重做，历史记录步数≥50 步
7.主题系统：内置≥5 套官方默认主题，一键切换全局样式
3.2 模板管理模块（需求 1/2/3）
3.2.1 模板创建功能（优先级：P1）
•用户可将当前编辑完成的思维导图保存为自定义模板
•保存时支持配置：模板名称、模板分类、模板备注、自动生成缩略图
•模板完整保存内容：节点层级结构、节点类型配置、全局主题样式、备注默认格式
3.2.2 模板增删管理（优先级：P1）
•模板库分为「官方默认模板」与「我的自定义模板」两类
•支持新增自定义模板、删除自定义模板；官方模板不可删除
•模板列表支持按名称、创建时间排序，支持关键词搜索
3.2.3 模板化新建（优先级：P1）
•新建思维导图时提供两个入口：使用默认模板快速创建、从模板库选择创建
•模板库展示所有模板的缩略图与名称，hover 显示详情
•支持用户设置任意模板为「默认新建模板」
3.3 自定义节点类型模块（需求 4/5，# 特色创新功能）
3.3.1 节点类型自定义配置（优先级：P2，特色创新）
•支持用户创建、编辑、删除多种节点类型，类型全局复用
•单种节点类型可配置项：
￮基础标识：类型名称、唯一 ID
￮视觉样式：节点图标（内置图标库 + 支持上传自定义图标）、节点形状、背景色、边框样式、文字格式
￮内容模板：节点默认文本、备注默认 Markdown 模板
•节点类型配置随思维导图 JSON 文件一同存储，他人打开文件时可完整展示所有自定义节点效果，无需额外配置
3.3.2 节点类型应用（优先级：P1）

•支持对已有节点切换类型，切换后样式与默认内容自动更新
•节点类型支持批量应用，框选多个节点可统一设置类型
3.3.3 内容模板继承（优先级：P1）
•创建指定类型节点时，自动填充该类型预设的节点文本与备注模板内容
•用户可在模板基础上自由编辑修改，不影响原类型配置

3.4.1 Markdown 实时渲染（优先级：P0，必须实现）
•节点备注区域全面支持 Markdown 语法，实现所见即所得实时渲染，输入过程中即时格式化，无需手动切换渲染模式
•支持的 Markdown 语法范围：
￮文本格式：各级标题、粗体、斜体、删除线、行内代码、引用块、分割线

•支持备注窗口独立放大预览，关闭后回到主编辑界面
3.4.3 节点文本自动换行（优先级：P1）
•节点文本内容超出节点默认宽度时，自动换行展示
•节点高度随文本行数自适应调整，保证内容完整显示不截断
•支持用户手动拖拽调整节点宽度，调整后文本自动重排换行
