## 版本定位

v1.2.0 是基础编辑体验增强版本，在不改变 `.lmind` 文件结构、不引入云同步和不破坏 v1.1.1 UI 信息架构的前提下，补齐节点复制 / 剪切 / 粘贴、框选节点、快捷键增强和拖拽调整父子层级基础版。

本版本继续坚持：

- 纯本地运行。
- 不上传用户文件或节点数据。
- 不引入用户登录、云同步或协同编辑。
- 不执行第三方插件 JavaScript。
- `.lmind` 基础结构保持兼容。

## 新增功能

### 节点复制 / 剪切 / 粘贴

- 支持复制当前主选中节点。
- 支持复制多个选中节点。
- 复制内容包含完整子树：节点文本、remark、children、nodeTypeId、collapsed 和可选 position。
- 支持剪切当前主选中节点或多个选中节点。
- root 中心主题不能剪切；多选中包含 root 时自动跳过 root。
- 支持 Ctrl+C / Ctrl+X / Ctrl+V。
- 支持节点右键菜单复制、剪切、粘贴为子节点。
- 支持画布右键菜单粘贴到中心主题和清空内部剪贴板。
- 粘贴时会为所有粘贴节点重新生成 ID，避免重复 ID。
- 粘贴和剪切粘贴进入撤销 / 重做历史。
- 内部剪贴板只保存在运行时状态中，不写入 `.lmind`。

### 复制为同级节点

- 支持 Ctrl+D 复制为同级节点。
- 非 root 节点会复制到同一父节点下。
- root 节点会复制为 root 的普通子节点，不覆盖中心主题。
- 新节点和完整子树都会生成新 ID。
- 操作进入撤销 / 重做历史。

### 框选节点

- 在画布空白区域按下并拖动，可显示半透明框选矩形。
- 框选覆盖到的节点会被选中。
- 不按 Shift 时，框选结果替换当前选择。
- 按住 Shift 框选时，命中节点追加到当前选择。
- 框选命中支持当前画布缩放和位移。
- 节点拖拽、右键菜单、顶部工具、左侧工具栏和备注面板不会触发框选。
- 框选状态不保存到 `.lmind`。

### 快捷键增强

- Ctrl+A：全选当前导图所有节点。
- Esc：优先关闭弹窗，其次关闭右键菜单，再取消框选，最后清空选择到中心主题。
- 快捷键帮助弹窗补充当前支持的快捷键。
- 快捷键帮助可从左侧工具栏或工具菜单打开。
- input、textarea、select 和 contenteditable 中不会误拦截浏览器默认复制、剪切、粘贴、全选行为。

### 拖拽调整父子层级基础版

- 普通节点拖拽仍保留自由移动 position 的能力。
- 拖拽节点到另一个有效节点附近时，目标节点显示可放置高亮。
- 松开鼠标时，如果释放点命中有效目标，被拖节点会移动为目标节点的子节点。
- root 中心主题不能被移动为其他节点的子节点。
- 不允许移动到自己或自己的子孙节点下，避免产生循环结构。
- 移动后节点 ID 不变化，text、remark、nodeTypeId、collapsed、position 和 children 不丢失。
- 拖拽调整层级进入撤销 / 重做历史。
- 拖拽目标高亮状态不保存到 `.lmind`。

### 备注面板布局修复

- 右侧备注面板顶部拆分为标题行和模式按钮行。
- “收起备注”不再和“放大预览 / 编辑模式 / 预览模式”挤在同一行。
- 当前节点标题单独占一行，不再被顶部按钮覆盖。

## root 节点保护

- root 可以复制，但粘贴后只能作为普通子节点副本。
- root 不能剪切。
- root 不能被拖动为其他节点的子节点。
- 批量删除、全选后删除、剪切、拖拽层级调整都会保留 rootNode。

## 数据兼容与安全

- 不修改 `.lmind` 文件结构。
- 不保存内部剪贴板状态。
- 不保存框选状态。
- 不保存快捷键帮助弹窗状态。
- 不保存拖拽目标高亮状态。
- JSON 导入 / 导出继续保留 `position`。
- 旧 `.lmind` 文件仍可正常打开。
- 复制粘贴、拖拽调整层级都会避免重复 ID 和循环 children。
- 所有文件读取、保存、导入、导出仍使用浏览器本地文件能力。
- 不上传用户文件、备注、模板、插件配置或操作数据。

## 本地运行

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

通常打开：

```text
http://localhost:5173/
```

## 已知限制

- 拖拽调整层级仍是基础版，不包含兄弟节点排序、批量层级拖拽和复杂智能排版。
- 框选后暂不包含批量移动和批量样式编辑增强。
- 本版本不包含 PDF 导出、云同步、用户登录、协同编辑或插件市场。
- 自动化测试主要覆盖纯函数和核心导入导出逻辑，完整浏览器交互仍需按验收清单手动回归。
- Vite 构建可能提示主 chunk 超过 500 kB，这是当前体积提醒，不是构建错误。
- Tauri 桌面端配置保留，但本次不发布桌面安装包。
