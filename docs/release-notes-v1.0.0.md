## 核心功能

- 基础思维导图编辑：新建、子节点、同级节点、删除、双击编辑。
- 左到右布局：根节点在左侧，子节点向右展开，连线随布局更新。
- 本地文件：保存和打开 `.lmind`，文件内容为标准 JSON。
- Markdown 备注：编辑、实时预览、预览模式和独立放大预览。
- 导入导出：Markdown、Excel、JSON、PNG、JPG，启用 TXT 插件后支持 TXT 导出。
- Excel 列映射：支持有表头和无表头 Excel 的手动列映射导入。
- 查找替换：支持节点文本、备注内容和全部范围。
- 模板系统：我的自定义模板和 8 个官方默认模板。
- 自定义节点类型：图标、形状、颜色、字号、加粗、默认文本和默认备注。
- 主题系统：内置 5 套主题，并支持插件主题包。
- 插件管理系统基础版：本地 JSON 安装、启用、禁用、卸载，不执行第三方 JS。
- 撤销 / 重做：历史记录不少于 50 步。
- 画布操作：缩放、平移、一键居中、折叠 / 展开。
- 性能测试工具：支持生成 100 / 500 / 1000 节点测试导图并复制结果。
- 自动化测试基础：Vitest 覆盖核心纯函数逻辑。

## 本地运行

```bash
npm install
npm run dev
```

默认在浏览器打开：

```text
http://localhost:5173/
```

## 构建方式

```bash
npm run build
```

构建产物输出到 `dist/`。

## 测试方式

```bash
npm run test
```

当前测试覆盖 Markdown 导入导出、JSON 校验、Excel 导入核心解析、TXT 导出、插件 manifest 校验、官方模板和性能测试数据生成。

## 数据安全说明

- 工具纯本地运行，不需要登录。
- `.lmind`、Markdown、Excel、JSON、图片和 TXT 文件均通过浏览器本地文件能力处理。
- 模板和插件注册表保存在浏览器 localStorage。
- 不上传用户文件。
- 不上传节点文本、备注内容、插件配置或操作数据。
- 插件基础版不执行第三方 JavaScript，不使用 `eval` 或 `new Function`。

## 已知限制

- 桌面端真实安装包尚未交付，当前仅完成打包准备文档和信创适配说明。

- Vite 构建可能提示主 chunk 超过 500 kB，这是体积提醒，不影响构建通过。
- 复制 / 剪切 / 粘贴、拖拽调整层级、框选批量操作仍属于后续功能。
- 插件系统基础版仅支持数据驱动扩展，不支持第三方动态代码执行、插件市场或云端下载。
