# v1.1.2 Draft Release Notes

## 版本定位

v1.1.2 是 v1.1.2-remark-ui 分支上的 UI 小修版本，只优化右侧备注面板顶部布局，不进入 v1.2 功能开发。

## 本次变更

- 右侧备注面板顶部第一行调整为左侧 `Remark`、右侧简洁折叠按钮。
- `放大预览 / 编辑模式 / 预览模式` 调整为第二行，避免和折叠按钮挤在同一行。
- 当前节点标题保持单独一行，并保留长标题自动换行能力。
- 备注面板折叠后显示窄条按钮 `‹ 备注`，减少画布侧边占用。

## 兼容性说明

- 不改变备注功能逻辑。
- 不改变 `.lmind` JSON 数据结构。
- 不影响 Markdown 编辑、实时预览、预览模式和放大预览。
- 不影响右侧备注面板折叠 / 展开。
- 不涉及 v1.2 功能和插件系统新增开发。

## 验证命令

```bash
npm run build
npm run test
```
