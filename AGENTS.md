# AGENTS.md

## 项目目标
实现一个纯本地、离线运行、跨平台的思维导图工具。

## 核心原则
1. 不允许上传用户文件或用户数据。
2. 所有数据保存为本地 .lmind JSON 文件。
3. 优先实现 P0 和 P1 需求。
4. 插件系统属于 P2，核心功能稳定后再做。
5. 每次改代码都必须说明改了什么。
6. 每次完成功能都必须运行测试。

## 开发方式
每次任务开始前：
1. 先阅读 docs/requirements.md。
2. 再阅读 docs/requirement-matrix.md。
3. 先给实现计划，不要直接乱写代码。

每次任务结束后：
1. 列出修改文件。
2. 列出测试命令。
3. 说明哪些需求已经完成。
4. 说明哪些还没完成。
